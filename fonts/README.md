# Put your fonts here

Drop the font files you want to proof into this folder. These formats all work:

- `.otf` and `.ttf` — regular desktop fonts
- `.ttc` — font collections
- `.woff` and `.woff2` — web fonts
- Variable fonts and color fonts in any of the above

Subfolders are fine if you like to keep families organized.

**Your fonts stay yours and stay here.** This folder is ignored by git, so nothing you put in it is ever committed, uploaded, or shared by this project.

When your files are in place, open Claude Code in this repo and run `/init-font`.
